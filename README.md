Vulnerabilidad Web: Explotación de SQL Injection (SQLi) en el panel de login mediante sqlmap, extrayendo credenciales de la base de datos users.

Acceso Inicial: Uso de credenciales obtenidas (pepe:P123pepe3456P) para ganar acceso vía SSH.

Escalada de Privilegios:

Identificación de binarios con permisos SUID (ls y grep).

Lectura del archivo /root/pass.hash utilizando el privilegio de grep.

Cracking de hash MD5 con CrackStation para obtener la contraseña de root: spongebob34.
